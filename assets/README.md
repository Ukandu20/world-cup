# Assets

Store non-code project media here.

## Structure

- `logos/countries/`: country logos, crests, or badges.
- `logos/world-cup/`: official tournament branding and logo files.
- `visualizations/`: generated visual assets for reports or app views.
- `charts/downloaded/`: externally downloaded charts and reference graphics.
- `charts/generated/`: charts produced by project scripts.

## Notes

- Keep filenames predictable and lowercase where practical.
- Prefer SVG or PNG for logos and chart exports.
- Only add official FIFA branding if you have the right to use and distribute it.
